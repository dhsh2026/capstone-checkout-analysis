# Checkout Experience Optimization: Growth Analytics

## Project Overview
This project analyzes an A/B test for a new checkout experience (**Variant B**). The goal is to determine if Variant B reduces drop-offs and increases revenue compared to the control (**Variant A**).

## Key Deliverables
- **ETL Pipeline:** Automated Python scripts to clean and transform raw event data.
- **Statistical Analysis:** Chi-Square testing for conversion lift and HTE (Segment) analysis.
- **Financial Modeling:** 30-day impact projections (Worst/Base/Best case).
- **Tableau Dashboards:** 4-part visualization of the user journey and experiment results.
